二、switch條件式

switch($[變數名稱]){
    case "[條件式A]":
         echo "結果A"
    break;
    case "[條件式B]":
         echo "結果B"
    break;
    case "[條件式C]":
         echo "結果C"
    break;   
    default:
	    echo "上述條件均不符合之結果"
}

	結果A
}else if([條件式B]){

	結果C
}else if([條件式D]){  
	結果D
}else{  
	上述條件均不符合之結果
}
